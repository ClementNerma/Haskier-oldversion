
VAMPP
=====

VAMPP est un outil permettant de créer un serveur pour partager des fichiers avec d'autres ordinateurs.

Mise en service
---------------

Une fois VAMPP installé, ouvrez un terminal et saisissez

```HTS (Haskier Terminal Script)
mkdir /webroot
vampp
```

Partage de fichiers
-------------------

Pour partager un fichier, placez-le dans le dossier *webroot*. N'importe quel visiteur pourra alors y accéder.
Pour créer une page web avec une mise en forme, créez un fichier nommé *xxx.xms* où *xxx* est un nom que vous pouvez choisir par vous-même.
Pour plus d'informations, reportez-vous à la documentation XMS de CenturyCode.
